# Team Name:- Console.io
# Hackathon:- Hackniche 2023
## Position:- Only Offline Shortlisting Cleared
#### Organizer:- Synapse, D.J. Sanghvi, Vile Parle, Mumbai

#### Description: A Website For Hosting and Participating in Hackathons. Users can create or join Hackathons. The Website, also, provides features to auto-shortlist teams for the Hackathon with the help of a ML Model and certain parameters mentioned by the Hosts.
